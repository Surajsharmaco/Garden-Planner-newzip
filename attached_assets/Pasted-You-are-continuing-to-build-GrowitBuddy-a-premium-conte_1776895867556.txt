AUTHORITY AUDIT PAGE (/authority-audit)
The most complex page. An 8-question interactive tool. No backend — all logic is inside AuthorityAudit.tsx.

8 Questions (in order):
1. Primary platform (LinkedIn / Instagram / YouTube / Twitter-X / Newsletter / Podcast / Multiple)
2. Role (Founder / Freelancer / Coach or Consultant / Content Creator / Executive / Other)
3. Posting frequency (Daily / 3-4x week / 1-2x week / A few times a month / Rarely)
4. How long posting (Under 3 months / 3-6 months / 6-12 months / 1-2 years / 2+ years)
5. Content type (Educational / Personal stories / Case studies / Opinions / Mixed)
6. Biggest challenge (Not enough reach / No inbound leads / Inconsistent posting / Content doesn't convert / No clear niche / No time)
7. Content goal (Build brand awareness / Generate leads / Become industry voice / Grow audience / Land speaking or press)
8. Ideal client/audience (text input)

Outputs:
- Authority Stage: Early Builder / Building Momentum / Stalled Authority / Plateau Stage / Unfocused Authority
- Named Content Gap: Distribution Gap / Trust Gap / System Gap / Positioning Gap / Conversion Gap / Niche Gap / Consistency Gap
- #1 Priority Action (specific to their answers)
- Personalized Plan (narrative prose, not bullet points — references their specific platform and role)

Intro page layout (two stacked options):
1. TOP (white card, black border): "Start the Audit yourself" — free, self-serve quiz with 4 deliverable cards inside + "Start the Audit" button
2. BOTTOM (dark black card): "Get your audit from GrowitBuddy's team" — links to /contact with "Book a Call" button on the right
